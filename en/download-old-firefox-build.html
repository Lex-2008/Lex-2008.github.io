Download old Firefox build

Created: 2017-04-28 — modified: 2017-04-29 — tags: Firefox links

deb packages for all versions starting with 3.5

If you happen to use Debian-based Linux and want to install an older (unsupported) version of Firefox for some reason (for compatibility with Selenium or Java), you can use this repository:

https://sourceforge.net/projects/ubuntuzilla/files/mozilla/apt/pool/main/f/firefox-mozilla-build/
