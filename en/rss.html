If someone wants to be notified about new posts in my blog via RSS, they can use a simple RSS generator

I was able to find two sites which worked for me and didn't require registration:

  • fivefilters.org: feed link The issue with this is that in free version it generates a feed that has only five (latest modified) items.
  • feedmaker.fly.dev: feed link The issue with this is that it generates items without date stamps, so they all will be edited "right now".

Pick your poison
