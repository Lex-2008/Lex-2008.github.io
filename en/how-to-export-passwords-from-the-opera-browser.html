How to export passwords from the Opera browser

Created: 2016-08-24 — modified: 2016-12-11 — tags: opera

In case if you want to use notepad as your password manager

That's pretty easy:

  1. Open password manager

  2. In each line, hover password field and click "show"

  3. Open javascript console (Ctrl+Shift+C) and paste this:

    document.querySelectorAll('#saved-passwords-list .deletable-item').forEach(a=>console.log(a.querySelector('a').innerText + ' <' + a.querySelector('a').href+'> ['+a.querySelector('.name input').value+'] ['+a.querySelector('.password input').value+']'))

It will dump all your shown passwords down in the javascript console.

Then you need to paste this into some decent text editor (like vim or sublime), which supports UNIX line endings and block editing, clean up and save.
