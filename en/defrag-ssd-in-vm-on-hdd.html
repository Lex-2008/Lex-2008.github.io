Defrag SSD in VM on HDD

Created: 2021-11-27 — modified: 2021-11-29 — tags: windows

Even if OS sees a storage device as SSD and refuses to defrag it - you might know better

Why

When host OS has at least one SSD disk, VMware reports all disks as SSD to guest OS, even if its disk image file is actually located on HDD.

How

Solution is to use a program released before SSD disks became widespread - for example, JkDefrad: official website, download link.
