Short memo about a good program

Thanks to my dear friend Craig Comstock, today I've learned about rlwrap - a readline wrapper program for Linux. Basically, it gives any program you like the power of readline.

So now, if a vendor of your favourite program doesn't want to add readline support to it, there is a solution!
